Use a TradingView embed if possible, or Plotly for a custom chart.Design: Clean, dark-themed UI. Mobile-responsive.Why this prompt works:Specific Math: It tells the Agent exactly how to calculate the zones so it doesn't guess.Weighted vs. Simple: It specifies the WMA for the 200-week and SMA for the 200-day, which is critical for your specific strategy.Actionable: It includes the "Action Instruction," so the app tells you exactly how much to spend on your Questrade account.Pro Tip for Replit:If the Agent asks for an API key, tell it: "Please use the DIA or CoinGecko public API which does not require an API key for basic price data."
